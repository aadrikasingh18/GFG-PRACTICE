<h2><a href="https://practice.geeksforgeeks.org/problems/operating-an-array/1?page=3&category[]=Arrays&category[]=Strings&category[]=Linked%20List&sortBy=difficulty">Operating an array</a></h2><h3>Difficulty Level : Basic</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size:18px">Given an array <strong>A</strong> of size <strong>N</strong>, your task is to do some operations, i.e., search an element <strong>x</strong>, insert an element <strong>y </strong>at index <strong>yi</strong>, and delete an element <strong>z</strong> by completing the functions. Also, all functions should return a boolean value.</span></p>

<p><span style="font-size:18px"><strong>Note:&nbsp;</strong></span></p>

<ul>
	<li><span style="font-size:18px">In delete operation return&nbsp;<strong>true</strong>&nbsp;even if element is not present.</span></li>

</ul>

<p><strong><span style="font-size:18px">Input Format:<br>

</span><span style="font-size:18px">5
</span><span style="font-size:18px">2 4 1 0 6
</span><span style="font-size:18px">1 2 2 0
</span><strong>

</span><span style="font-size:18px">1 1 1</span></pre>

<p><strong><span style="font-size:18px">Your Task:</span></strong><br>
<span style="font-size:18px">Since this is a function problem, you only need to complete the provided functions.</span></p>
